# MERN-Demo

RESTful web application

---
## Stack

- NestJS
- NodeJS
- MongoDB
- React
---
## Features

- [x] Token based auth
- [x] Protect routes | Roles
- [x] Seed for mongodb
- [x] Dockerize
- [ ] Loading indicator
- [ ] Backend pagination
- [ ] Unit testing
- [|] Search by name ,email or id
---
### Prerequisites

- Docker Engine
- Docker Compose
---
## Run


```sh
cd mern-demo
docker compose up
```

###### Modifications

```sh
cd mern-demo
docker compose build
docker compose up
```
